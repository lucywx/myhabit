# MyHabit 用户试用指南

## 🎯 试用目标
帮助您快速了解MyHabit的功能，并提供宝贵的使用反馈。

## 🚀 快速开始

### 1. 注册账号
- 访问应用首页
- 点击"注册"标签
- 填写用户名、邮箱和密码
- 密码要求：至少8位，包含大小写字母、数字和特殊字符

### 2. 设置目标
- 登录后自动跳转到目标设置页面
- 输入您想要养成的习惯
- 设置惩罚金额（可选）
- 选择惩罚类型：平台或朋友

### 3. 每日打卡
- 在首页点击"今日打卡"
- 上传照片证明
- 完成打卡

### 4. 查看进度
- 在用户中心查看打卡记录
- 查看目标完成情况
- 查看转账记录

## 📱 核心功能

### 目标管理
- 设置个人习惯目标

- 查看目标进度

### 打卡系统

### 惩罚机制
- 错过打卡自动转账

### 邀请系统
